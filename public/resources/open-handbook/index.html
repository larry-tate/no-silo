    About

    The Open Handbook

    The Open Handbook project is a free, open-source reference handbook that introduces students to the conventions of academic writing and research. The project was originally written in LaTeX. I am in the process of translating the project into .md files so that I can include it on this website.

    If you would like to collaborate on the project, please contact me. If you know your way around git, you can fork the project on GitHub.
